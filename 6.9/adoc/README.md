# Meveo

Meveo is a platform that allow to develop and execute enterprise back and front applications. It is intended to be run by Wildfly 15 with full Jakarta EE stack.

[Visit us at https://meveo.org](https://meveo.org)

## Building Meveo Documentation

Convert the AsciiDoc to HTML5 by invoking the `process-resources` goal (configured as the default goal):

```console
 mvn
 ```

Open the file _target/generated-docs/index.html_ in your browser to see the generated HTML file.
